Nesta fase de desenvolvimento do Fixr algumas telas foram criadas para acompanhar o respectivo desenvolvimento. São estas:

## Tela de cadastro de ordem de serviço
A tela de cadastro de ordens de serviço, foi previamente criada para abrigar o botão que aciona o cadastro de clientes, visando a facilidade de se criar o cadastro de um cliente caso não haja, no momento da criação da ordem de serviço. O formulário definitivo das ordens de serviço será desenvolvido nos proximos semestres

## Tela de cadastro de clientes
A tela de cadastro de clientes podem ser acessadas na seção [cadastros](/OAuth_e_cliente/cadastros/cadastro_cliente).
