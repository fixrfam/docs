No quarto semestre foi criado o cadastro de clientes, e previamente o cadasto de ordens de serviço. Essenciais para o inicio de qualquer gestão e gerenciamento de ordens de serviço.

## Cadastro de clientes
O cadastro de clientes permite que se crie um cadastro para a ordem de serviço pertencer a um cliente em específico, permitindo controle e consistência nas ordens de serviço. Sendo facilitado atráves de um drawer presente na tela de cadastro de ordem de serviço.  O formulário possui algumas validações de dados para melhorar a consistência dos dados cadastrados, como verificações de tamanho de strings ou inteiros para determinado campo e verificações de existencia de cadastro, evitando duplicações.
