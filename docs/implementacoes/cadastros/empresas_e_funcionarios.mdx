No terceiro semestre também foram criados os cadastros de empresas e de funcionários. Essenciais para o inicio de qualquer gestão e gerenciamento de ordens de serviço.

## Cadastro de empresas
O cadastro de empresas permite que se crie um perfil empresarial de administrador, onde é possivel criar os cadastros e perfis de funcionários, gerenciar estoques, ordens de serviço, clientes etc. O formulário possui algumas validações de dados para melhorar a consistência dos dados cadastrados, como verificações de tamanho de strings ou inteiros para determinado campo e verificações de existencia de cadastro, evitando duplicações. Para isso, algumas informações são coletadas, como mostra a imagem abaixo.
![Hero](/public/images/tela_cadastro_empresa.png)

## Cadastro de funcionários
A partir de um perfil administrador é possivel cadastrar os perfis de funcionários. Onde é possivel além de criar o acesso já denominar o cargo daquele perfil. O que já direciona o permissionamento de acesso ao sistema mas os usuários não-administradores. As mesmas verificações existentes para o cadastro de empresas se aplicam para o cadastro de funcionários. A imagem abaixo mostra o formulário de cadastro de funcionários.
